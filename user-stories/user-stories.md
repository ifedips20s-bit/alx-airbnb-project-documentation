# User Stories

Below are the core user stories that capture the main interactions in the Airbnb Clone backend system.

---

### 1. User Registration
**As a user, I want to create an account so that I can access the Airbnb platform and use its features.**

### 2. User Login
**As a user, I want to log in securely so that I can manage my bookings and account.**

### 3. Search for Properties
**As a guest, I want to search for available properties by location and date so that I can find a place that suits my needs.**

### 4. Book a Property
**As a guest, I want to book a property so that I can reserve a place to stay for my trip.**

### 5. Make Payment
**As a guest, I want to make secure payments for my booking so that my reservation can be confirmed.**
